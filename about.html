MY STRENGTHS
  • Design & Development

    디자인의 감성을 이해하고 개발의 논리를 존중하며,
    그 사이의 간극을 메우는 완성도 높은 코드를 지향합니다.

  • User Experience

    화려한 기능 이전에 사용자의 편리함을 먼저 생각하며,
    웹 표준을 기반으로 한 접근성 높은 인터페이스를 구축합니다.

  • Maintainable Code

    한 번 쓰고 버려지는 코드가 아닌,
    누구나 읽기 쉽고 관리하기 편한 구조적인 마크업을 위해 고민합니다.

  • Problem Solving

    브라우저 간의 차이를 좁히고 기술적 제약을 창의적으로 해결하며,
    동료와 함께 최선의 결과물을 만들어내는 과정에 집중합니다.

  • Growth

    빠르게 변하는 웹 환경 속에서도 본질을 놓치지 않으며,
    더 효율적인 구현 방법을 찾기 위해 끊임없이 역량을 확장합니다.

Stack

  • HTML

    75%

    시맨틱 마크업과 명확한 구조 설계를 기반으로,
    유지보수성과 확장성이 높은 HTML 코드를 작성할 수 있는 수준의 역량을 갖추고 있습니다.

  • CSS

    75%

    Flex와 반응형 레이아웃을 기반으로 한 구조적인 스타일링에 능숙하며,
    가상요소와 트랜지션을 활용해 시각적인 완성도를 높이는 CSS를 작성합니다.

  • JavaScript

    50%

    DOM 제어와 이벤트 기반 UI 기능 구현에 능숙하며,
    인터랙션 요소를 직접 설계해 자연스러운 사용자 경험을 만드는 JavaScript 코드를 작성합니다.
    다양한 라이브러리(AOS, Swiper, GSAP)를 사용합니다.

  • jQuery

    60%

    jQuery를 활용해 DOM 제어와 다양한 UI 기능을 구현하며,
    구조적인 인터랙션을 안정적으로 제작할 수 있습니다.

  • Github

    50%

    GitHub에서 프로젝트를 관리하고,
    commit·push·pull 및 GitHub Pages 배포를 안정적으로 수행할 수 있는 기본 역량을 갖추고 있습니다.

  • Figma

    25%

    Figma를 통해 화면 요소의 간격과 레이아웃 정보를 확인하며,
    개발에 필요한 디자인 스펙을 파악하는 용도로 활용할 수 있습니다.

Timeline

1차 PORTFOLIO

신세계 L&B와 한솔PNS를 처음으로 만들며 기본기를 다졌고,
그 과정에서 배운 점들을 바탕으로 첫 포트폴리오 사이트를 완성했습니다.
혼자 작업하다 보니 디자인보다는 구현에 집중했고,
‘포인트 색감만이라도 잘 잡자’는 생각으로 진행한 초기 시도였습니다.

바로가기

2차 PORTFOLIO

르스페이스와 기아 홈페이지를 추가로 클론코딩하며 퍼블리싱 역량을 더욱 확장했고,
이를 바탕으로 기존과는 다른 방식의 포트폴리오 입장 경험을 만들었습니다.
인트로 페이지에는 ‘입장’ 패스워드를 입력해야 메인 페이지로 이동할 수 있는 기능을 구현했으며,
메인에서는 다양한 라이브러리를 활용해 더욱 풍부한 인터랙션을 시도했습니다.
개발 과정에서도 “조금 더 새로운 느낌은 없을까?”를 고민하며 구성과 연출을 지속적으로 발전시켰습니다.

바로가기

3차 PORTFOLIO

신세계 L&B를 처음 클론코딩했지만 부족함을 느껴 학습 후 완전히 새로 제작해 완성도를 높였습니다.
이후 삼성 라이온즈 홈페이지를 리뉴얼하며 구조와 디자인 분석 능력을 키웠고,
대기업 페이지도 경험해보고 싶어 제네시스 홈페이지를 클론코딩했습니다.
이번 포트폴리오에서는 새로운 방식을 시도하고자 사이드 네비게이션을 도입하고,
날씨 API를 활용한 위젯 기능도 직접 구현하며 사용자 경험을 확장했습니다.
색상도 차분한 색상으로 바꿨습니다.

바로가기

4차 PORTFOLIO

JavaScript 실력이 부족하다고 느껴 간단한 게임들을 직접 만들며 학습을 했습니다.
카드 게임은 이미지가 섞인 뒤 잠시 공개되었다가 다시 뒤집히는 구조로 구현했고,
숫자 야구는 랜덤 정답 생성과 S/B/Out 판정 로직을 직접 작성해 완성했습니다.
포트폴리오 사이트 디자인 면에서는 메인 컬러가 배경에 묻히는 문제를 해결하기 위해
여러 사이트를 참고했고,
기본 색상은 흰색·검정, 메인 컬러는 포인트로 사용해 조화를 맞추었습니다.
또한 기존에 사용하던 사이드 메뉴 구조를 유지하면서 펼쳐지는 네비게이션 UX도 개선했습니다.

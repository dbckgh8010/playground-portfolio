어려웠던 점 & 해결 과정

문제

처음 사용하는 날씨 API를 활용하여 원하는 지역의 날씨 정보를 가져오고 화면에 출력하는 과정이 가장 어려웠습니다. API 데이터 구조를 이해하고 필요한 정보를 추출하는 데 시간이 소요되었습니다.

해결

공식 문서와 예제를 참고하며 API 요청 및 응답 구조를 분석하였고, 필요한 데이터를 가공하여 화면에 출력할 수 있도록 구현했습니다. 반복적인 테스트를 통해 오류를 수정하며 기능을 완성했습니다.

1차 PORTFOLIO

신세계 L&B와 한솔PNS를 처음으로 만들며 기본기를 다졌고,
그 과정에서 배운 점들을 바탕으로 첫 포트폴리오 사이트를 완성했습니다.
혼자 작업하다 보니 디자인보다는 구현에 집중했고,
‘포인트 색감만이라도 잘 잡자’는 생각으로 진행한 초기 시도였습니다.

2차 PORTFOLIO

르스페이스와 기아 홈페이지를 추가로 클론코딩하며 퍼블리싱 역량을 더욱 확장했고,
이를 바탕으로 기존과는 다른 방식의 포트폴리오 입장 경험을 만들었습니다.
인트로 페이지에는 ‘입장’ 패스워드를 입력해야 메인 페이지로 이동할 수 있는 기능을 구현했으며,
메인에서는 다양한 라이브러리를 활용해 더욱 풍부한 인터랙션을 시도했습니다.
개발 과정에서도 “조금 더 새로운 느낌은 없을까?”를 고민하며 구성과 연출을 지속적으로 발전시켰습니다.

3차 PORTFOLIO

신세계 L&B를 처음 클론코딩했지만 부족함을 느껴 학습 후 완전히 새로 제작해 완성도를 높였습니다.
이후 삼성 라이온즈 홈페이지를 리뉴얼하며 구조와 디자인 분석 능력을 키웠고,
대기업 페이지도 경험해보고 싶어 제네시스 홈페이지를 클론코딩했습니다.
이번 포트폴리오에서는 새로운 방식을 시도하고자 사이드 네비게이션을 도입하고,
날씨 API를 활용한 위젯 기능도 직접 구현하며 사용자 경험을 확장했습니다.
색상도 차분한 색상으로 바꿨습니다.

4차 PORTFOLIO

JavaScript 실력이 부족하다고 느껴 간단한 게임들을 직접 만들며 학습을 했습니다.
카드 게임은 이미지가 섞인 뒤 잠시 공개되었다가 다시 뒤집히는 구조로 구현했고,
숫자 야구는 랜덤 정답 생성과 S/B/Out 판정 로직을 직접 작성해 완성했습니다.
포트폴리오 사이트 디자인 면에서는 메인 컬러가 배경에 묻히는 문제를 해결하기 위해
여러 사이트를 참고했고,
기본 색상은 흰색·검정, 메인 컬러는 포인트로 사용해 조화를 맞추었습니다.
또한 기존에 사용하던 사이드 메뉴 구조를 유지하면서 펼쳐지는 네비게이션 UX도 개선했습니다.

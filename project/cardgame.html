어려웠던 점 & 해결 과정

문제

사용자가 같은 카드를 연속으로 클릭하거나 여러 장을 빠르게 클릭하면 카드 상태가 꼬이는 현상이 발생.

해결

첫번째 카드, 두번째 카드, lockBoard 변수를 활용하여 한 번에 두 장만 선택할 수 있도록 제한하고, 비교가 끝날 때까지 추가 입력을 막아 안정적으로 동작하도록 개선.

개선 & 배운 점

  • 이벤트 제어 경험

  • CSS 애니메이션 활용

  • 상태 관리의 중요성

  • 배열 데이터 활용
